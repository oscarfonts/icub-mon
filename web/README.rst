Setup Virtualenv
----------------

::

 sudo pip install virtualenv
 virtualenv icub_env
 source icub_env/bin/activate


Install dependencies
--------------------

 ::

 pip install simplejson
 pip install psycopg2
 pip install Flask
 pip install Flask-SQLAlchemy
 pip install Flask-Restless


Run debug
---------

 ::

 python icub-mcm.py
